# 📊 Atividade 06 - Power BI | APEX 2º Bimestre

Projeto desenvolvido para a disciplina de APEX com o objetivo de aplicar conceitos de transformação de dados, DAX e criação de dashboards no Power BI.

## 🚀 Objetivos da Atividade

- Importar um dataset `.csv` no Power BI
- Aplicar transformação de dados utilizando Linguagem M
- Criar fórmula utilizando DAX
- Desenvolver um dashboard interativo
- Aplicar filtros para análise dos dados

## 🛠️ Tecnologias Utilizadas

- Power BI
- Linguagem M
- DAX
- Dataset CSV

## 📌 Funcionalidades Desenvolvidas

- Tratamento e transformação de dados
- Criação de métricas utilizando DAX
- Dashboard para análise visual dos dados
- Aplicação de filtros interativos
